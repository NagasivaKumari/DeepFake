# Project Documentation

## Overview
This project is a comprehensive admin dashboard with advanced features such as analytics, multi-language support, user management, and backend security enhancements.

## Features
1. **Advanced Analytics Dashboard**: Visualize key metrics using charts and graphs.
2. **Multi-Language Support**: Switch between English and Spanish seamlessly.
3. **User Management**: Manage users with role-based access control and bulk actions.
4. **Backend Security**: Includes rate limiting, HTTPS enforcement, and host restrictions.
5. **Frontend Performance Optimization**: Lazy-loaded components for faster load times.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/NagasivaKumari/DeepFake.git
   ```
2. Navigate to the project directory:
   ```bash
   cd DeepFake
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## API References
